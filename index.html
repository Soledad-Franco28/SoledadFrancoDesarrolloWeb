Objetivos

Al finalizar el curso se espera que las y los estudiantes puedan:

  • Crear paginas web utilizando HTML,CSS,SVG y JavaScript.
  • Escribir el 100% del código de un sitio web.
  • Diseñar una web que se adapte a diferentes tamaños de pantalla y gestionar la navegacion.
  • Crear elementos animados que interactúen con el usuario.
  • Consumir API´s con JavaScript para colocar contenidos en un sitio.

Contenido

UNIDAD 1: HTML Y Desarrollo Web

Entorno de desarrollo. Editor de código. Inspector y herramientas de desarrollo de los navegadores. Funcionamiento básico de un sitio web. etiquetas y atributos. Etiquetas de texto. Imagenes y elaces. Etiquetas de bloque. Etiquetas de documentos. repositorios y git.

UNIDAD 2: Introducción a la CSS

Vincular un CSS, Sinexis básica de CSS.Selectores. Estilos y tipografías. Estructura en las hojas de estilo. Selectores avanzados. Estilo de navegación. Rests CSS. El modelo de la caja. Diplays. Floats. Positions. Estilos para listas. Border, border-radius y box-shadow. Fondos e imagenes de fondos. WebFonts. Colores y opacidad. Bordes y redondeados. Sombras. Desgrades y fondos.
